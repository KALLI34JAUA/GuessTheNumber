# GuessTheNumber
Java number guessing game project
Java Number Guessing Game

A simple Java-based Number Guessing Game where the user tries to guess a randomly generated number within a limited number of attempts. This project demonstrates basic Java programming concepts such as loops, conditional statements, input/output handling, and random number generation.

Features

Random Number Generation: The system generates a random number within a specified range.

User Input: Players can input their guesses through the console.

Hints: The game provides feedback whether the guess is too high, too low, or correct.

Limited Attempts: Optionally, users can have a limited number of attempts to increase difficulty.

Score Tracking: Optionally track the number of attempts taken to guess correctly.

Replay Option: Players can restart the game after finishing.

Technologies Used

Programming Language: Java

IDE: Eclipse, IntelliJ IDEA, or any Java-supported IDE

Java Version: Java 8 or higher

Installation

Clone or download the repository.

Open the project in a Java IDE (Eclipse, IntelliJ, NetBeans).

Compile the Java file (GuessingGame.java) using the IDE or command line:

javac GuessingGame.java


Run the compiled program:

java GuessingGame

Usage

Run the program.

Enter your guess when prompted.

Follow the hints (“Too high” or “Too low”) until you guess the correct number.

The program will display the number of attempts taken.

Choose to play again or exit the game.
